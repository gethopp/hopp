The Hopp desktop app provides a Settings panel to customize your call, camera, audio, and miscellaneous preferences.

<Image src={settings} class="rounded-md" alt="Hopp settings panel" />

## Call

**Call feedback popup** — Show a feedback popup when a call ends. Default: on.

**Show dock icon when in call** (`show_dock_icon_in_call`) — Keep the dock icon visible during a call. Default: on.

## Camera

**Start camera on call start** — Automatically open the camera when a call starts. Default: off.

## Audio

**Start microphone on call start** (`start_mic_on_call`) — Automatically unmute the microphone when a call starts. Default: on.

## Shortcuts

Global keyboard shortcuts for controlling calls from anywhere on your system. All shortcuts are customizable in the Settings window.

| Action              | macOS         | Windows / Linux |
| ------------------- | ------------- | --------------- |
| Toggle microphone   | `Cmd+Shift+A` | `Ctrl+Shift+A`  |
| Toggle camera       | `Cmd+Shift+V` | `Ctrl+Shift+V`  |
| Toggle screen share | `Cmd+Shift+S` | `Ctrl+Shift+S`  |
| End call            | `Cmd+Shift+W` | `Alt+Q`         |

## Miscellaneous

**Custom Backend URL** — Override the default backend server. Leave empty to use the default Hopp backend. This is useful for [self-hosted Hopp instances](/open-source/self-hosting).
