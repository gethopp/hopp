The Hopp desktop app provides a Settings panel to customize your call, camera, audio, and miscellaneous preferences.

<Image src={settings} class="rounded-md" alt="Hopp settings panel" />

## Call

**Call feedback popup** — Show a feedback popup when a call ends. Default: on.

**Show dock icon when in call** — Hide the dock icon to save space when in a call. Default: on.

## Camera

**Start camera on call start** — Automatically open the camera when a call starts. Default: off.

## Audio

**Start microphone on call start** — Automatically unmute the microphone when a call starts. Default: on.

## Miscellaneous

**Custom Backend URL** — Override the default backend server. Leave empty to use the default Hopp backend. This is useful for [self-hosted Hopp instances](/open-source/self-hosting).
